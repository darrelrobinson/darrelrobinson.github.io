Running, saving, and exporting several models with four lines of code

Darrel Robinson

July 2, 2018

I run into this kind of situation a lot, I have a single model specification that I want to apply to several different dependent variables.

Letâ€™s use as an example the mtcars dataset. There are 11 variables in the dataset:

data <- mtcars
dim(data)
## [1] 32 11
names(data)
##  [1] "mpg"  "cyl"  "disp" "hp"   "drat" "wt"   "qsec" "vs"   "am"   "gear"
## [11] "carb"
m1 <- lm(cyl ~ mpg, data = mtcars)
m2 <- lm(disp ~ mpg, data = mtcars)
m3 <- lm(hp ~ mpg, data = mtcars)
m4 <- lm(drat ~ mpg, data = mtcars)
m5 <- lm(wt ~ mpg, data = mtcars)

m7 <- lm(vs ~ mpg, data = mtcars)
m8 <- lm(am ~ mpg, data = mtcars)
